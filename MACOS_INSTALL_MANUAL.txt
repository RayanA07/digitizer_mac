DATA DIGITIZER 2.11.MAC INSTALL MANUAL

This package is for macOS Tahoe.

You do not need Python.
You do not need VS Code.
You do not need Homebrew.
You do not need to install Tesseract.


OPTION A - EASIEST INSTALL

1. Open Safari.

2. Go to:

   https://github.com/RayanA07/digitizer_mac/releases/tag/v2.11.mac

3. Download:

   DataDigitizer-2.11.mac-Installer.app.zip

4. Open Downloads in Finder.

5. Double-click DataDigitizer-2.11.mac-Installer.app.zip.

6. Right-click or Control-click DataDigitizer Installer.app.

7. Click Open.

8. If macOS warns about an unidentified developer, click Open again.

9. Click Install in the installer window.

10. Wait for the progress bar to finish.

11. Click Close.

The installer copies the apps to:

~/Applications/Data Digitizer

It also creates Desktop launchers for:

- Digitizer.app
- AccuracyTester.app
- DataDigitizer CLI.command


OPTION B - ZIP PACKAGE

1. Download DataDigitizer-2.11.mac-macOS.zip.

2. Double-click the ZIP in Downloads.

3. Open the extracted folder.

4. Right-click or Control-click DataDigitizer Installer.app.

5. Click Open.

6. Click Install.


IF MACOS BLOCKS THE APP

If macOS says the app cannot be opened:

1. Open System Settings.

2. Go to Privacy & Security.

3. Scroll down.

4. Click Open Anyway for the blocked app.

If macOS says the app is damaged, open Terminal and run:

xattr -dr com.apple.quarantine "$HOME/Downloads/DataDigitizer Installer.app"

If the app is already installed and blocked:

xattr -dr com.apple.quarantine "$HOME/Applications/Data Digitizer/Digitizer.app"
xattr -dr com.apple.quarantine "$HOME/Applications/Data Digitizer/AccuracyTester.app"
xattr -dr com.apple.quarantine "$HOME/Applications/Data Digitizer/datadigitizer"


HOW TO OPEN THE APP

Double-click Digitizer.app on the Desktop.

To keep it in the Dock:

1. Open Digitizer.app.

2. Right-click or Control-click the Dock icon.

3. Click Options.

4. Click Keep in Dock.


CLI QUICK START

Open Terminal and run:

"$HOME/Applications/Data Digitizer/datadigitizer" 'digitizer_cli(pic_dir="/Users/yourname/Downloads/Example 2.png", output_dir="/Users/yourname/Downloads/testcli")'

Auto mode:

"$HOME/Applications/Data Digitizer/datadigitizer" 'digitizer_cli(pic_dir="/Users/yourname/Downloads/Example 2.png", , , , output_dir="/Users/yourname/Downloads/testcli")'

Manual mode:

"$HOME/Applications/Data Digitizer/datadigitizer" 'digitizer_cli(pic_dir="/Users/yourname/Downloads/Example 2.png", color=(255,0,0), tick_setting=([10,200],[500,200],[10,200],[10,20]), axis_values=(0,10,0,100), output_dir="/Users/yourname/Downloads/testcli")'


HOW TO COPY A MAC FILE PATH

1. Open Finder.

2. Find the graph image.

3. Right-click or Control-click the image.

4. Hold the Option key.

5. Click Copy "<filename>" as Pathname.

6. Paste it between the double quotes after pic_dir=.


CLI OUTPUT

The CLI prints something like:

Pipeline done. Output data to /Users/.../Example 2_digitized_points.csv. Points: 201.
Overlapping plot: /Users/.../Example 2_digitized_overlay.png

It creates:

- a CSV file with digitized points
- a PNG overlay showing the placed points
